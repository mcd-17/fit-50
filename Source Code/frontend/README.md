# Homestick Client Side

This project is the client side of the Homestick application.

## Project Commands

### Setup

Install Package:

```bash
npm install
pnpm install
yarn install
```

Run the application
```bash
npm run dev
pnpm run dev
yarn dev
```

Build the application
```bash
npm run build
pnpm run build
yarn build
```

Running the Production Build
```bash
npm start
pnpm start
yarn start
```
